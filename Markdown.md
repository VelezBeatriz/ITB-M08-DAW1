### **2. Texto en negrita, cursiva y tachado**

- **Negrita:** Usa `**` o `__`.  
  ```markdown
  **Texto en negrita** o __Texto en negrita__
  ```

- *Cursiva:* Usa `*` o `_`.  
  ```markdown
  *Texto en cursiva* o _Texto en cursiva_
  ```

- ~~Tachado:~~ Usa `~~`.  
  ```markdown
  ~~Texto tachado~~
  ```

Resultado:

**Texto en negrita**  
*Texto en cursiva*  
~~Texto tachado~~

---


Resultado:

- Elemento 2
  - Sub-elemento 2.1
  - Sub-elemento 2.2

### **9. Tareas**

Usa `- [ ]` para tareas pendientes y `- [x]` para tareas completadas.

```markdown
- [x] Tarea 1 (completada)
- [ ] Tarea 2 (pendiente)
```

Resultado:
- [x] Tarea 1 (completada)  
- [ ] Tarea 2 (pendiente)
